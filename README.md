Glowing Leaves (English)

This is a Blender scene for scattering glowing leaf models using a particle system. 

Glowing green leaves float and spread in space, creating a fantastical effect. 

-Test Environment- 

Blender Version: 3.4 

Rendering Engine: Cycles 

-How to Use- 

  1. Download and open Glowing Leaves.blend.

　2. Try rendering the video!

-Base File- 

This scene is based on a tutorial file.

The leaf models, particle settings, materials, and visual adjustments were all created and modified by me.
   
-Rendering Settings-

Rendering Engine: Cycles

Sampling: 128 

Denoicing: None
